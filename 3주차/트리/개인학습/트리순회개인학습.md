preorder('A')        ← A 방문 (루트 출력)
│
├── preorder('B')    ← 왼쪽 자식 B 방문
│   ├── preorder('.')  ← B의 왼쪽 자식 없음 → return
│   └── preorder('.')  ← B의 오른쪽 자식 없음 → return
│
└── preorder('C')    ← 오른쪽 자식 C 방문
    ├── preorder('.')  ← C의 왼쪽 자식 없음 → return
    └── preorder('.')  ← C의 오른쪽 자식 없음 → return



      A
     / \
    B   C
